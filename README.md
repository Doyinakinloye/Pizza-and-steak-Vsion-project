🍕🥩 *Pizza-and-Steak Vision Project*

**Overview**

This project demonstrates the use of Convolutional Neural Networks (CNNs) for food image classification, focusing on distinguishing between pizza and steak. Built and tested in Google Colab, it leverages curated datasets from the Food101 dataset.

**Features**

* End-to-end deep learning workflow in Colab notebooks.

* CNN-based architecture for image recognition.

* Training and evaluation on pizza vs. steak datasets.

* Clear, reproducible code for experimentation and learning.

**Getting Started**

* Clone the repository:

bash
git clone https://github.com/Doyinakinloye/Pizza-and-steak-Vsion-project.git

* Open the provided Colab notebooks:

Pizza-Steak Vision.ipynb

* Run cells step-by-step to train and evaluate models.

**Dataset**

Images are sourced from the Food101 dataset, a curated collection designed for rapid prototyping and scalable experiments.

* Classes: Pizza and Steak.

**Requirements**

* Python 3.x

* TensorFlow / Keras

* Google Colab environment

**Results**

* Training Accuracy: ~95% after 10 epochs

* Validation Accuracy: ~92%

* Confusion Matrix: Shows strong separation between pizza and steak classes, with minimal misclassifications.
